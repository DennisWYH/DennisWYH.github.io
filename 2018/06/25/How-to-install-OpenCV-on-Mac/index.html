How_to_install_OpenCV_on_Mac

  |   124 parole   |   1 mins Posted on 2018-06-25 | In AI , Computer Vision

Step #1: Install XCode

Download Xcode from App Store
Accept the Apple developer license

$ sudo xcodebuild -license

Install the Apple Command Line Tools

$ sudo xcode-select --install

Step #2: Install Homebrew

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew update
update ~/.bash_profile

$ nano ~/.bash_profile

Step #3: Install Python 2.7 and Python 3 using Homebrew

$ brew install python python3
$ brew linkapps python3

Step #4: Install OpenCV 3 with Python bindings on macOS using Homebrew

$ brew tap homebrew/science
/ or
$ brew tap homebrew/core

$ brew install opencv3 --with-contrib --with-python3 --HEAD
I successed without the --HEAD part(This took me a whole morning)

Step #5: Testing

>>> python3
>>> import cv2
>>> '3.4.1-dev'
